
# ことね自主トレシミュレータ
模拟ことね的卡组以及学马仕游戏模式，并使用强化学习进行让KTN自己玩自己的卡组。
(WIP)

## play.py
直接自己玩，用于测试系统是否正常工作。

## train.py
训练模型

## play_with_model.py
使用训练好的模型进行游戏
